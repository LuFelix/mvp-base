🚀 SOLUÇÃO RÁPIDA (escolha uma):


  ⭐ OPÇÃO 1 - Script Automatizado (MAIS FÁCIL):

     $ ./migration-seed.sh

     Executa tudo em sequência:
     1. Verifica Docker
     2. Executa migrations (cria tabelas)
     3. Executa seed (popula dados)


  📦 OPÇÃO 2 - Comandos Manuais:

     # 1. Entrar no container
     $ docker-compose exec -it backend bash

     # 2. Executar migrations
     $ npm run typeorm migration:run

     # 3. Executar seed
     $ npm run seed:verbose

     # 4. Sair
     $ exit


  🔧 OPÇÃO 3 - Docker direto (sem entrar):

     $ docker-compose exec backend npm run typeorm migration:run
